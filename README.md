# cn-ssl-journal

本仓库用于归档和发布多个独立 HTML 页面，内容涉及与 SSL/TLS 相关的公开资料与记录。仓库不针对任何单个域名或具体网站，仅作为静态页面的集中存放与版本管理空间。

## 项目简介

cn-ssl-journal 是一个以 HTML 文件为主体的归档仓库，收录了若干独立页面，旨在长期保存与 SSL/TLS 相关的公开信息。所有内容均为静态页面，可部署于任何支持静态文件的托管服务。

## 目录说明

```
.
├── index.html          # 入口页面，列出所有归档页面的索引
├── pages/              # 存放各独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/             # 公共资源（可选）
│   ├── css/
│   └── js/
└── README.md           # 本文件
```

- `index.html`：仓库的主页，提供所有归档页面的导航链接。
- `pages/`：核心目录，每个文件对应一份独立的归档内容。
- `assets/`：存放页面共用的样式、脚本或图片等资源。

## 页面归档说明

- 每个 HTML 文件均包含完整的页面结构，可独立打开浏览。
- 归档内容主要来源于公开渠道，经整理后以静态页面形式保存。
- 页面内容可能随时间更新，历史版本可通过 Git 提交记录追溯。
- 不包含任何交互式后端功能，所有内容均为纯前端展示。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过 Pull Request 或直接提交完成。
